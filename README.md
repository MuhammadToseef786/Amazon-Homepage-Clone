# 🛒 Amazon Front Page Clone

A clean and responsive **Amazon Front Page Clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of the Amazon homepage while focusing on modern web design principles and responsive layouts.

## 🚀 Features

* Responsive design for different screen sizes
* Modern Amazon-inspired user interface
* Navigation bar with search section
* Hero banner
* Product category cards
* Footer section
* Clean and well-structured code

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🎯 Purpose

This project was created to improve my front-end development skills by practicing layout design, responsive web development, and clean code structure.
